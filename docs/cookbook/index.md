# Cookbook & Examples

Task-focused, step-by-step recipes for accomplishing common documentation-authoring goals with
folio.

## What Are Cookbook Recipes?

Each recipe follows a problem-solution format:

- **Problem** — what you're trying to accomplish
- **Solution** — step-by-step commands with explanations
- **Explanation** — how it works behind the scenes
- **Related** — links to the relevant commands, skills, or agents

Recipes include a level (Beginner/Intermediate/Advanced) and a time estimate.
