# scripts/config/exclusions.txt
# Shared exclusion config for folio count/staleness validators.
# Consumed via folio_exclusions / folio_is_excluded in scripts/lib/schema.sh.
#
# Format (one entry per line; same schema as craft's exclusions.txt):
#   filepath            — exclude entire file from all checks
#   filepath:pattern    — exclude specific pattern matches in that file
#                         (e.g. "docs/CHANGELOG.md:12 commands" skips that one
#                         historical count while the rest of the file is scanned)
#   dir/                — exclude entire directory (trailing slash)
#   # comment           — ignored
#   blank lines         — ignored
#
